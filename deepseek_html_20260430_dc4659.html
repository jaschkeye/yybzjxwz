📊 任务1 — 制作《质检点定义表》（WPS表格）
1

新建 WPS 表格，输入表头

依次点击单元格并输入：
A1 → 输入 项目类型
B1 → 输入 质检点
C1 → 输入 特征及判断

选中 A1:C1，按 Ctrl+B 加粗，点击工具栏「居中」。

📸 此时你的表格：
项目类型质检点特征及判断
2

填写有效性错误（17条）

A2 输入 有效性错误，然后选中 A2:A18 → 右键「设置单元格格式」→「对齐」→ 勾选「合并单元格」。

在 B2 到 B18 依次输入：
空旷音、回音 / 电流音 / 噪音过大 / 语音失真 / 喷麦严重 / 截幅、消波 / 音量过低 / 忽高忽低 / 直流偏移 / 半频、经音 / 部分缺能量 / 丢帧、跳帧 / 非母语、方言不符、口音错误 / 乱念、无法单独成句 / 中间停顿超过1秒 / 一人多录 / 读错、无法转写

在 C2 输入 无效判为有效，然后向下拖动填充柄（单元格右下角小方块）一直拖到 C18。

📸 此时你的表格（前几行）：
项目类型质检点特征及判断
有效性错误空旷音、回音无效判为有效
电流音无效判为有效
噪音过大无效判为有效
…………
3

填写截取错误（2条）

A19 输入 截取错误，选中 A19:A20 合并。

B19 输入 静音段截取过长，C19 输入 超过0.8秒算错误

B20 输入 静音段截取过短，C20 输入 低于0.2秒算错误

📸 此时你的表格（截取部分）：
项目类型质检点特征及判断
截取错误静音段截取过长超过0.8秒算错误
静音段截取过短低于0.2秒算错误
4

填写转写错误（6条）

A21 输入 转写错误，选中 A21:A26 合并。

依次填写：
B21: 多字、少字 → C21: 所听即所写
B22: 错字、别字 → C22: 避免同音字错误
B23: 数字转写错误 → C23: 数字写汉字，不写阿拉伯数字
B24: 语气词加口字旁 → C24: 通用规则
B25: 英语/拼写/缩略词 → C25: 单词间留空格，字母大写
B26: 符号缺失或多余 → C26: 成对符号完整

📸 此时你的表格（转写部分）：
项目类型质检点特征及判断
转写错误多字、少字所听即所写
错字、别字避免同音字错误
数字转写错误数字写汉字，不写阿拉伯数字
语气词加口字旁通用规则
英语/拼写/缩略词单词间留空格，字母大写
符号缺失或多余成对符号完整
5

美化并保存

选中整个表格区域（A1:C26），点击工具栏「边框」→「所有框线」。
调整列宽：鼠标放在 A/B/C 列之间双击自动适应。
保存为 质检点定义表.xlsx。

✅ 现在你的表格就是标准的质检点速查表，一共25条规则。
🔄 任务2 — 绘制质检流程图（EdrawMax）
1

启动 EdrawMax → 点击「新建」→ 选择「流程图」→ 选择「基本流程图」。

2

从左侧「符号库」中记住：
矩形 = 普通步骤
菱形 = 判断分支
圆角矩形 = 开始/结束

3

画第一阶段：标准制定

依次拖入：
圆角矩形（开始） → 矩形（解读客户规范） → 矩形（参考历史经验） → 矩形（编写质检点） → 菱形（客户确认？）。
用箭头连接。不通过→箭头指回修改环节；通过→箭头向下。

4

画第二阶段：人员培训

矩形（培训质检员） → 菱形（考核通过？）
不通过→箭头指向二次培训（加一个矩形），再指回菱形；通过→箭头向下。

5

画第三阶段：执行质检

矩形（分配质检任务） → 矩形（按规范质检） → 菱形（数据合格？）
不合格→箭头指向矩形（返工），再指回质检；合格→矩形（提交验收） → 圆角矩形（结束）。

6

双击每个形状输入文字，调整位置对齐。
点击「文件」→「导出」→ 保存为 PNG 图片。

💡 流程图完成后应包含三个明显阶段，有“是/否”分支和回流箭头。
